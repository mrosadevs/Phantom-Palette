> A sleek, beautifully animated color palette generator built for designers and developers.

## ✨ Features

- **10 Harmony Modes** — Random, Analogous, Complementary, Triadic, Split-Complementary, Monochromatic, Warm, Cool, Pastel, Vibrant
- **Lock Colors** — Pin any swatch while regenerating the rest
- **One-Click Copy** — Click any color card to copy the hex code instantly
- **Export Formats** — CSS Variables, SCSS, Tailwind Config, JSON
- **PNG Download** — Export your palette as a ready-to-share 1200×630 image
- **Keyboard Shortcut** — Hit `Space` to generate a new palette instantly
- **Fully Animated** — Smooth card transitions, glowing backgrounds, ripple effects
- **Responsive** — Works great on desktop and mobile

## 🎨 Harmony Modes Explained

| Mode | Description |
|------|-------------|
| **Random** | Fully random, high-saturation colors |
| **Analogous** | Colors adjacent on the color wheel |
| **Complementary** | Opposite hues for high contrast |
| **Triadic** | Three evenly-spaced hues |
| **Split-Complementary** | One base + two near-complements |
| **Monochromatic** | One hue, varying lightness |
| **Warm** | Reds, oranges, and yellows |
| **Cool** | Blues, teals, and purples |
| **Pastel** | Soft, high-lightness tones |
| **Vibrant** | Maximum saturation, bold colors |

## 📦 Usage

Just open `index.html` in a browser — or visit the live demo. No install needed.

```bash
git clone https://github.com/mrosadevs/Phantom-Palette.git
cd Phantom-Palette
open index.html
```
